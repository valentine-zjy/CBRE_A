# 房地产Agent项目

## 项目目标
为世邦魏理仕（CBRE）相关商业地产业务设计并逐步实现一套 Agent 系统，
用于提升研究、交易、招商租赁、资管、设施管理、项目管理、文档自动化等工作效率。

## 方法论
本项目采用 harness 思想推进：
- Agent != 单次问答
- Agent = Model + Harness
- Harness 包含：
  - 任务拆解
  - 状态管理
  - 工具调用
  - 知识/文件读写
  - 人工确认点
  - 日志追踪
  - 评测与验收

## 研究边界
聚焦商业地产，不做纯住宅C端导流。
优先覆盖：
- 交易/投资分析
- 招商租赁
- 估值/资管
- 物业/设施管理
- 项目管理
- 研究报告与文档自动化
- 楼宇运营

## 目录说明
- docs/: 项目文档与共享记忆
- prompts/: 各类 worker 的 prompt 契约
- src/: harness 与 workflow 代码
- data/: 样例任务、样例文档、评测集
- outputs/: 运行结果、trace、报告

## 开发原则
1. 先做可运行骨架，再加业务能力
2. 所有输出尽量落文件
3. 所有能力都应可测
4. 所有事实与假设分开
5. 缺失信息明确标注“未公开”或“待补充”
