# 业务Agent地图

本文件定义面向客户售前展示的 6 类业务 Agent。每类 Agent 都使用同一接口描述，直接回答三个问题：

1. 这类 Agent 服务谁。
2. 它在业务里具体做什么。
3. 它产出什么可复核结果。

## 六类 Agent 总览
| Agent | 服务对象 | 高频任务 | 代表结果物 |
| --- | --- | --- | --- |
| 研究 Agent | 研究分析师、顾问、市场研究团队 | 市场扫描、可比项目整理、研究简报初稿 | 市场摘要、可比项目清单、待确认事项 |
| 交易 Agent | 交易经理、投资分析师、尽调协同角色 | 管线跟踪、尽调推进、审批材料整理 | 状态看板、阻塞项清单、下一步建议 |
| 招商租赁 Agent | 招商经理、租赁经理、资产运营团队 | 线索分级、房源匹配、租约节点推进 | 线索优先级、推进清单、风险提醒 |
| 估值资管 Agent | 估值经理、资管经理、投后分析师 | 假设整理、情景分析、资产经营复盘 | 参数摘要、情景对比、估值支持说明 |
| 楼宇运营 Agent | 物业经理、设施经理、楼宇运营团队 | 工单协同、事件升级、服务闭环跟踪 | 工单队列、异常告警、运营摘要 |
| 文档报告 Agent | 分析师、项目经理、顾问团队 | 报告初稿、材料拼装、引用整理 | 结构化初稿、引用清单、复核清单 |

## 研究 Agent
- `agent_name`: 研究 Agent
- `target_role`: 研究分析师、顾问、市场研究团队
- `business_tasks[]`: 市场信息汇总、可比项目搜集、区域供需跟踪、研究简报初稿生成
- `inputs[]`: 市场报告、项目资料、历史研究文档、公开资讯、客户问题清单
- `agent_actions[]`: 检索资料、抽取关键事实、整理可比样本、生成结构化摘要、标记冲突和待确认项
- `outputs[]`: 市场摘要、可比项目清单、研究简报初稿、证据引用列表、待确认事项
- `business_value[]`: 缩短信息收集时间、提高研究输出一致性、减少遗漏来源
- `human_checkpoints[]`: 关键结论确认、对外引用审校、最终报告签发
- `non_goals[]`: 不替代正式研究判断、不生成无来源事实、不绕过人工发布

## 交易 Agent
- `agent_name`: 交易 Agent
- `target_role`: 交易经理、投资分析师、尽调协同角色
- `business_tasks[]`: 交易管线跟踪、尽调清单推进、审批节点管理、交易材料整理
- `inputs[]`: 项目管线、尽调材料、会议纪要、审批节点、外部往来记录
- `agent_actions[]`: 汇总状态、识别阻塞项、生成下一步待办、同步跨角色进度、整理投决材料
- `outputs[]`: 交易状态看板、阻塞项清单、任务分配建议、交易摘要、复核前材料包
- `business_value[]`: 提升项目推进透明度、减少跨团队漏项、降低材料整理成本
- `human_checkpoints[]`: 投资判断、关键商业条款确认、审批提交与签发
- `non_goals[]`: 不直接做买卖决策、不替代法务合规审批、不自动推进真实流程

## 招商租赁 Agent
- `agent_name`: 招商租赁 Agent
- `target_role`: 招商经理、租赁经理、资产运营团队
- `business_tasks[]`: 线索分级、租户需求匹配、沟通跟进、租约节点推进
- `inputs[]`: 线索池、房源信息、租户需求、沟通记录、租约台账、节点日志
- `agent_actions[]`: 识别高优先级线索、推荐跟进动作、整理租约状态、提示逾期和风险、沉淀沟通纪要
- `outputs[]`: 线索优先级列表、房源匹配建议、跟进计划、租约进度摘要、风险提醒
- `business_value[]`: 提高线索转化效率、减少跟进断点、帮助团队统一状态口径
- `human_checkpoints[]`: 商务谈判判断、关键租约条款确认、合同签署前复核
- `non_goals[]`: 不替代经纪判断、不自动承诺商务条件、不替代合同法务审阅

## 估值资管 Agent
- `agent_name`: 估值资管 Agent
- `target_role`: 估值经理、资管经理、投后分析师
- `business_tasks[]`: 估值假设整理、现金流与租约信息汇总、情景对比、经营复盘
- `inputs[]`: 资产台账、租约数据、现金流模型、预算预测、运营数据、估值模板
- `agent_actions[]`: 汇总关键参数、比较情景差异、抽取租约影响项、生成估值支持说明、沉淀资产复盘要点
- `outputs[]`: 参数摘要、情景对比表、估值支持说明、资产经营复盘摘要、待确认参数清单
- `business_value[]`: 降低手工汇总成本、提升分析可复核性、统一口径表达
- `human_checkpoints[]`: 关键参数确认、最终估值结论确认、对外披露审阅
- `non_goals[]`: 不替代专业估值模型、不替代投资委员会判断、不直接发布正式结论

## 楼宇运营 Agent
- `agent_name`: 楼宇运营 Agent
- `target_role`: 物业经理、设施经理、楼宇运营团队
- `business_tasks[]`: 工单分级、服务请求闭环、异常事件协同、运营日报周报整理
- `inputs[]`: 工单记录、设备状态、租户服务请求、SLA 规则、事件日志、运营台账
- `agent_actions[]`: 汇总待办、识别超时与异常、生成分派建议、沉淀事件摘要、提示升级路径
- `outputs[]`: 工单优先级队列、异常告警、服务闭环摘要、升级清单、运营摘要
- `business_value[]`: 提升工单闭环效率、减少跨团队沟通损耗、提高运营透明度
- `human_checkpoints[]`: 派单确认、供应商与预算决策、紧急事件指挥
- `non_goals[]`: 不直接控制设备、不自动触发真实派单、不替代现场管理责任

## 文档报告 Agent
- `agent_name`: 文档报告 Agent
- `target_role`: 分析师、项目经理、顾问团队
- `business_tasks[]`: 报告初稿、投决材料拼装、经营复盘文档整理、引用与附件归档
- `inputs[]`: 报表数据、模板、源文档、历史版本、图片附件、引用规则
- `agent_actions[]`: 抽取字段、拼装结构、生成初稿、挂接证据、标记缺失与冲突、输出复核清单
- `outputs[]`: 结构化初稿、引用清单、缺失字段列表、冲突提示、复核版结果物
- `business_value[]`: 缩短成稿时间、提高文档结构一致性、增强对外材料可追溯性
- `human_checkpoints[]`: 关键措辞审校、事实核对、正式签发与外发
- `non_goals[]`: 不直接输出终稿、不替代法务或品牌审校、不掩盖未确认信息

## 与 CBRE 业务线的映射
| 业务线或售前话题 | 主 Agent | 协同 Agent | Demo 中建议展示的结果物 |
| --- | --- | --- | --- |
| 市场研究与顾问输出 | 研究 Agent | 文档报告 Agent | 市场摘要、可比项目清单、研究简报初稿 |
| 交易推进与尽调协同 | 交易 Agent | 估值资管 Agent、文档报告 Agent | 状态看板、阻塞项清单、投决材料摘要 |
| 招商租赁与客户转化 | 招商租赁 Agent | 研究 Agent、文档报告 Agent | 线索优先级、跟进计划、租约状态摘要 |
| 估值与资产经营复盘 | 估值资管 Agent | 文档报告 Agent、楼宇运营 Agent | 参数摘要、情景对比、经营复盘说明 |
| 楼宇与设施运营 | 楼宇运营 Agent | 文档报告 Agent | 工单队列、异常告警、运营摘要 |
| 报告与材料自动化 | 文档报告 Agent | 研究 Agent、交易 Agent、估值资管 Agent | 结构化初稿、引用清单、复核清单 |
