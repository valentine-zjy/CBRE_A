# 竞品事实表

本文件定位为附录事实库，只记录公开可追溯信息，不承担我方产品主叙事。

## 使用规则
- 只写公开事实，不写结论先行的判断。
- 没有证据的字段统一标注为 `未公开` 或 `待补充`。
- “对我方意味着什么”的推断，统一放在 [07_竞品摘要附录.md](07_竞品摘要附录.md)。

## 字段说明
| 字段 | 说明 |
| --- | --- |
| 公司/产品 | 对外公开名称 |
| 主要覆盖环节 | 公开资料能确认的业务范围 |
| 对应我方 Agent | 与我方 6 类业务 Agent 的对应关系 |
| 已公开 AI/Agent 表述 | 只记录官方可验证说法 |
| 集成与连接能力 | 官方公开的系统连接、API 或数据整合能力 |
| 公开客户/案例 | 官方公开引用的客户或项目 |
| 价格公开情况 | 仅记录是否公开，不做估算 |
| 证据来源 | 官方页面或公开新闻稿 |
| 备注 | 未公开、待补充或范围限定 |
